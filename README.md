# Opcode Based Android Malware Detection Using Machine Learning

A bachelor thesis demonstrating the application of ML in malware detection. The aim to the project is to demonstrate the use of ML techniques and feature compression in malware classification.
The method used here is hash buckets sorting particular snippets of operational code. These snippets are copied from reverse-engineered .apk files with the help of Baksmali.
In the research part of the thesis more information about different combinations of features is described in detail.

## Usage

Download app.py and feature_extractor.py with the above model. Run app.py and provide a path to an .apk file to be scanned.
